# [MZ] Lim_MZ_UI Framework

안녕하세요! (안녕하세요~) 기분이 너무 좋아서 코드에 하트를 찍을 뻔한, **행복증 말기** 프로그래머 **Lim**입니다! 

이 라이브러리는 제가 일본 DLsite 상업 서클과 Steam 프로젝트를 진행하면서, "어떻게 하면 RPG Maker의 UI를 더 쫀득하고 부드럽게 만들 수 있을까?"라는 고민 끝에 엔진의 렌더링 레이어를 직접 재구성해서 만든 결과물이에요. 

---

### Core Features

단순히 예쁜 스킨을 씌우는 게 아니에요. 저는 엔진의 '속살'을 만지는 걸 정말 좋아하거든요!

* **Rendering Extension**: 
    - `Bitmap` 클래스를 확장해서 베지어 곡선(Bezier)과 라운드 렉트(Round Rect)를 지원합니다.
    - 고성능 알파 블렌딩(BltAlpha)과 다각형 마스킹 기능을 통해 뽀얗고 미려한 화면을 구성할 수 있어요.
* **Math Logic Engine**: 
    - 가중치 기반 랜덤 알고리즘, 고정밀 팅김 방지(Anti-bounce) 애니메이션 등 게임 로직에 꼭 필요한 수학 도구들을 꽉꽉 채워 넣었어요.
* **Architecture Design**: 
    - 단순히 '돌아가는 코드'는 조금 부끄럽잖아요? 인터랙션 로직과 표현층을 완전히 분리해서, UI의 반응 속도가 정말 쫀득쫀득해지는 것을 느끼실 수 있을 거예요!

---

### Development Philosophy

제 코드는 언제나 기술 그 자체보다 **당신의 마음**을 먼저 읽으려고 노력해요.

1. **진짜 필요한 게 뭘까?**: 클라이언트분이 "이런 게 하고 싶어!"라고 말씀하시면, 그게 어떤 장면에서 쓰일지, 유저가 어떤 기분을 느낄지 먼저 고민해서 코드를 짜요.
2. **합리적인 솔루션**: 무조건 어렵게 짜는 건 하수! 상황에 맞춰서 가장 가볍고 튼튼한 길을 찾아드릴게요. 제가 좀 '행복증'이라 긍정적인 방향으로 최적화를 참 잘하거든요!
3. **다정한 코드**: 나중에 이 코드를 보실 분이 "와, Lim이 정말 예쁘게 짜놨네?"라고 느끼실 수 있게 깔끔한 구조와 친절한 주석을 약속해요.
